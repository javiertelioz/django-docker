# Django - Docker (python:3.5-alpine, phpmyadmin, mysql, nodejs)

### Install Django

```sh
$ docker-compose exec web sh
$ sh /code/bin/install.sh
```

### Admin Data

> ADMIN_USERNAME: admin
> ADMIN_EMAIL: admin@myproject.com
> ADMIN_PASSWORD: yourpassword

# Gulp

### Developer
```sh
$ docker-compose exec web sh
$ gulp watch
```
### Production

```sh
$ docker-compose exec web sh
$ gulp
```
